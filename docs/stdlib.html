Loom Standard Library

Built-in functions and methods available in the Loom language.

Global Functions

Function Returns Description
print(...)NonePrints values to stdout.
input(prompt)strReads a line from stdin.
clear()NoneClears the terminal screen.
read(file)ResultReads file content.
write(file, txt)ResultWrites content to file.
ls(path)ResultLists directory entries.
exists(path)boolChecks if path exists.
sleep(ms)NonePauses execution.
env(key)str | NoneGets environment variable.
json(str)ResultParses JSON string.
net.connect(ip, port)ResultTCP connection test.

Data Conversions

Function Purpose
str(val)Converts value to string.
int(val)Converts value to 64-bit integer.
float(val)Converts value to 64-bit float.
bool(val)Converts value to boolean.

Type-Specific Methods

Strings

  • s.len(): Length of string.
  • s.replace(old, new): Replace occurrences.
  • s.split(delim): Split into List.
  • s.starts_with(pre) / s.ends_with(suf)

Lists

  • l.len(): Item count.
  • l.push(item): Add to end.
  • l.map(func): Transform items.
  • l.filter(func): Filter items.

Maps

  • m.len(): Entry count.
  • m.keys() / m.values()
