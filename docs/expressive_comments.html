Expressive Comments in Loom

Loom also features a unique "Expressive Comment" system that makes your code more scannable, self-documenting, and easier to maintain.

The Comment Types

Prefix Type Purpose
//*SectionLogic block labels or major boundaries.
//!UrgentCritical warnings or important constraints.
//?QueryDebug thoughts or TODO questions.
//.MetaMetadata (Author, Version, etc.).
//;LegalLicensing or copyright headers.

Example Usage

//; loom auth system | licensed under mit
//* user identity

frame User { username: str, id: int }

act login(u, p) {
    //? should we implement rate limiting?
    let hash = hash_pass(p) //! never log or leak this value
}
